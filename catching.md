# Catching 
## Spawning 
Pokémon spawn based on message activity, which is 24 valid messages. Messages that classified as valid needs to be visible to Pokétwo, 1.5 seconds apart from the same users, 1 second apart from the server, not a command, and not a message from a bot.

## Catching
To catch a Pokémon you would need to guess the Pokémon species name through `@Pokétwo#8236 catch/c <Pokémon>`, failed attempt does not make the Pokémon despawn. However, the Pokémon may despawn if the spawning process is re-trigger.
![Successfully catching a Pokémon.](https://placehold.co/800.png)

### Hints
If you do not know the name of the Pokémon, Pokétwo will reveal 50% of the name through `@Pokétwo#8236 hint/h`. Cool down applies 50s per channels, and 60s per users.

### Milestones
Catching a specific amount of Pokémon will give Pokécoins based on the table referred below:

| Caught  | Pokécoins |
| ------- | --------- |
| 1       | 35        |
| 10      | 350       |
| 100     | 3,500     |
| 1,000   | 35,000    |
| 10,000  | 350,000   |
| 100,000 | 3,500,000 |

## Quests
Quests in Pokétwo are categorized in two category, regional quests, and type quests. To check the progress of the quest use `@Pokétwo#8236 quests/q`.

 Completing a quest will give Pokécoins and once entirety finished a respective badge.

| ------ | -------- |

| 50     | 5,000PC  |
| 100    | 10,000PC |
